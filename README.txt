TCP/XML Клиент-серверное приложение написано с помощью платформы .NET Framework 4.8 на основе инструментов для создания классических Desktop-приложений Windows Forms.

Инструкция по запуску:
1. Запустите серверную часть ServerTCP.exe по пути ServerTCP - bin - Release 
2. Запустите клиентскую часть ClientTCP.exe по пути ServerTCP - bin - Release
3. Переведите серверную часть в активное состояние соответствующей кнопкой для подключения.
4. Выберите XML-файл на клиентской стороне, а затем отправьте запрос на сервер для получения результатов (На сервере отображаются все детали информации о подключаемых клиентах, получаемых XML-файлов и их целостное содержание в соответствии с корректным форматом)

*К архиву прилагается два эталонных файла XML для взаимодействия с сервером и листинг программы в виде скриншотов.
